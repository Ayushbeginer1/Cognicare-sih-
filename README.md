# Cognicare

A comprehensive digital platform offering mental health and psychological support for students in higher education.

## Introduction
Cognicare empowers students by providing accessible mental health resources, confidential assessments, and direct counselor support. The platform is designed to promote well-being, resilience, and privacy.

## Features
- Anonymous mental health assessments
- Self-help resource library
- Real-time chat support with counselors
- Progress tracking and personalized recommendations
- Secure, confidential data management

## Getting Started
1. Register or log in as a student or counselor.
2. Complete assessments and explore resources.
3. Connect with counselors via secure chat for support.

## Technologies Used

- React.js
- MongoDB
- Socket.io
